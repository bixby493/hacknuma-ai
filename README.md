## 🧠 What is Hacknuma AI?

Hacknuma AI is a multi-functional AI web application designed to combine powerful tools into a single dashboard.

It integrates:
- AI chatbot  
- Developer tools  
- Productivity features  
- Automation concepts  

👉 Everything in one place — simple, fast, and effective.
