# 🚀 Hacknuma AI

A powerful AI backend + frontend system built with Node.js and Groq API.

## ⚡ Features
- AI Chat System (Groq API)
- Backend (Express.js)
- Environment-secured API keys
- Frontend UI (HTML + JS)
- Scalable architecture (apps + workers)

## 🧠 Tech Stack
- Node.js
- Express.js
- Groq API
- HTML / JS
